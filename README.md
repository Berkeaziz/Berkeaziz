## Selected Projects

### Real-Time Crypto Analytics Platform

A real-time data engineering project that processes cryptocurrency trade data using streaming technologies.

**Stack:** Kafka, PySpark Structured Streaming, PostgreSQL, Redis, FastAPI, Grafana, Docker

Key features:

- Real-time trade ingestion from Binance WebSocket
- Kafka-based streaming architecture
- PySpark Structured Streaming for transformation and aggregation
- PostgreSQL warehouse for historical storage
- Redis for low-latency latest candle access
- Grafana dashboards for real-time monitoring

---

### Energy Market Analytics

A machine learning and data engineering project focused on electricity market forecasting and decision support.

**Stack:** Python, Pandas, LightGBM, Airflow, MLflow, PostgreSQL, Streamlit, Docker

Key features:

- Electricity price forecasting
- Time series feature engineering
- Model training and evaluation pipeline
- Automated workflows with Airflow
- Experiment tracking with MLflow
- Dashboarding for model outputs and evaluation

---

### Global Air Quality Data Platform

An end-to-end ELT data platform for collecting, transforming, and analyzing air quality data.

**Stack:** Python, PostgreSQL, Snowflake, dbt, Airflow, Docker

Key features:

- Data extraction from OpenAQ API
- Raw data storage in PostgreSQL
- Snowflake-based analytical warehouse
- dbt models for staging and marts
- Star schema design
- Automated data workflows
